# **App Name**: Media Muse

## Core Features:

- Caption Suggestion: Suggests captions for uploaded images using AI.
- Music Suggestion: Suggests background music for uploaded images using AI.
- Image Upload: Allows users to upload images to the application.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey creativity and sophistication.
- Background color: Light blue (#E8EAF6), a very desaturated version of the primary, provides a gentle backdrop to make the images pop.
- Accent color: Violet (#7E57C2), analogous to the primary, brings a magical feel to interactive elements and CTAs.
- Font: 'Inter', a sans-serif font that offers a clean and modern aesthetic, for both headlines and body text.
- Use clean, simple icons to represent actions like generating, saving, and sharing.
- A minimalist layout with a focus on the image display area.
- Smooth transitions and subtle loading animations to enhance the user experience.
